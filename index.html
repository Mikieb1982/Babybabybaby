Summary of Aesthetic Changes: * Font: Changed to "Kalam" with adjusted base size and line height for better readability. Applied to inputs as well. * Backgrounds: Main content area uses a soft off-white (bg-off-white via stone-50) for subtle contrast with the body's paper texture. Results area matches this. * Borders: Main container keeps border-2, but internal elements (inputs, results box, buttons) use thinner border or specific border colors. Dashed lines used for dividers. * Shadows: Added shadow-sm to main container and primary button for subtle depth. * Controls: Slightly refined radio button appearance and dropdown arrow. Inputs have clearer focus states. * Spacing: Adjusted padding, margins, gaps, and text sizes throughout for better visual hierarchy and breathing room. * Buttons: Standardized button appearance using .btn classes, added subtle active state scaling. Used icons more consistently for Save/Remove/Copy actions. This version aims for a more refined, layered, and visually balanced take on the "paper and drawn" theme.
